# Data

Raw datasets are not stored in this repo — they're large public
datasets held in AWS S3 (`s3://payce-data-platform/raw/`).

## Sources (all from Kaggle)
- PaySim — https://www.kaggle.com/datasets/ealaxi/paysim1
- IEEE-CIS Fraud Detection — https://www.kaggle.com/c/ieee-fraud-detection
- Home Credit Default Risk — https://www.kaggle.com/c/home-credit-default-risk
- LendingClub — https://www.kaggle.com/datasets/wordsforthewise/lending-club

Download via the Kaggle CLI, then upload to S3.
